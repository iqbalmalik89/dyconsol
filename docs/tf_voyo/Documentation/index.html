Overview

This template is built on Twitter Bootstrap framework and it uses jQuery, a javascript library to handle some of the actions throughout the template, and in order to be able to easily work with the template, you need to know at least the basics of these two frameworks, but it is not necessary.

Also, we used some third party plugins in this template like isotope, owl-carousel, flexslider (we will enumerate them later in this documentation ) and there may be some situations where you will need or want to make some changes. e.g. If you want some charts in your site, you'll need to initialize the chart.js plugin, and therefore you have to know how to do that. But, don't worry, we will explain what every plugin does and where you could get more informations about them. So, let's get started.

Files Structure

Once downloaded, unzip the compressed folder to see the structure of the template. inc folder includes all third party plugins while css, js and php folders includes only files created by me.

We strongly advise you not make your customizations in these files, as you may need to replace these files in the future updates. If you want to add something to your site, or if you want to change something (styles or scripts), please put your code in separated files (e.g. user_style.css).

Section Structure - Full-width with background

For a faster development I created a script that let you set a background image to any section through an HTML5 data attribute. So, if you want a background section, you'll have to add a custom-bg class and a data-bg attribute, pointing to your image path. Also, if you want a "parallax" section (image positioned fixed, not really parallax), just add a parallax class. Make sure to choose a proper image (at least 1920x1080) for parallax background. bg-layer element is optional. See example below.

<section class="section custom-bg parallax" data-bg="assets/images/yourImage.jpg">
    <div class="bg-layer"></div>
    <div class="container">
        <div class="row">
            <div class="col-sm-12">

                ---   content   ---

            </div>
        </div> 
    </div> <!-- END Container -->
</section> <!-- END Services Section -->

Slider

For this template I chose to use flexslider plugin. I customized it a little, adding support for animations, fixed images etc. If you want a slider with animations, just add an animated class to any element inside the slider and a data-anim attribute where you specify the name of the animation. Also, you could delay the animation, adding a data-delay attribute. The value for this attribute must be a number greater than 0 (it is measured in milliseconds - e.g. 200, 500, 1100 etc). You could choose to link images through data-bg attribute, using the custom-bg class, or you can use the css method. See an example below.

<section id="main-slider" class="flexslider">
    <ul class="slides">
        <!-- Slide 1 -->
        <li class="custom-bg" data-bg="assets/images/yourImage.jpg">
            <div class="bg-layer"></div>
            <div class="flex-content">
                <div class="container">
                    <div class="row">
                        <div class="col-sm-12">
                            <div class="animated" data-anim="fadeInUp" data-delay="800">
                                ---   content   ---
                            </div>
                        </div>
                    </div> 
                </div> <!-- END Container -->
            </div> <!-- END flex content -->
        </li> <!-- END Slide 1 -->
        <!-- Slide 2 -->
        <li class="custom-bg" data-bg="assets/images/yourImage.jpg">
            <div class="bg-layer"></div>
            <div class="flex-content">
                <div class="container">
                    <div class="row">
                        <div class="col-sm-6">
                            <div class="animated" data-anim="bounceInRight" data-delay="200">
                                ---   content   ---
                            </div>
                        </div>
                        <div class="col-sm-6">
                            <div class="animated" data-anim="bounceInLeft" data-delay="500">
                                ---   content   ---
                            </div>
                        </div>
                    </div> 
                </div> <!-- END Container -->
            </div> <!-- END flex content -->
        </li> <!-- END Slide 2 -->
    </ul> <!-- END Slides -->
</section>

Elements

There are tons of elements where you can choose from! Whether you want alerts, animations or maps, you can create the page yourself by including any information you want.

How to?

You know how copy-paste work, right? For example, if you want a button, you will have to open the elements_buttons.html page and look for the desired button. It is better if you use a web developer tool ( Firebug or Chrome Developer Tools ) to easily locate an element.

Layout Type

This template allows you to choose between boxed and full width layout. The default layout is full width. So, if you want boxed layout, just add the layout-boxed id to the body tag. Bear in mind that you need to do this task for every page! See an example below:

<body id="layout-boxed">
   
    -- content --

</body>

Colors

If you want a different color than the default one, you just need to add a link to the css file after that main style. Bear in mind that you need to do this task for every page! Replace color-blue with the favourite color style from assets/css/colors/ folder.

<head>
    
    -- some styles --

    <link rel="stylesheet" href="css/style.css">
    <link rel="stylesheet" href="css/colors/blue.css">  -- here --

</head>

OR, you can create your own color by changing one variable in the css/color_scheme.less file and then compiling this less file into a css file (I will explain later how to do that)

// Change #22aba6 with your favorite color
@color: #22aba6;

Let's say your favourite color is rgb(52,119,220). Open color_scheme.less file and change @color to rgb(52,119,220).

// Change #22aba6 with your favorite color
@color: rgb(52,119,220);

Now you have to go to less2css.org and copy all the code from this less file in the less section ( left ), then copy all the code from css section ( right ) and paste it to color_scheme.css file and save it. Now you have to link color_scheme.css to your pages. Bear in mind that you need to do this task for every page!

<head>
    
    -- some styles --

    <link rel="stylesheet" href="css/style.css">
    <link rel="stylesheet" href="css/color_scheme.css">  -- here --

</head>

That's it!

Patterns

If you chose to have boxed layout, you can set a background pattern ( image or color ) to the body. So, in order to do that, you have to add a background property to the body (As I said at the begining, please create a new css file and put your new styles there). You will find 8 patterns in images/patterns. See an example below:

/* Background Pattern (Or image) */

body {
    background: url('../images/patterns/1.png');
}

/* Background color */

body {
    background: rgb(34,171,166);
}

Static Menu?

This template allows you to position your menu to top (fixed) or you can change it to be static. The default behavior for this template is fixed menu, but if you want static menu all you need to do is to add static-menu class to the body tag.

For short: 

<body class="static-menu">

Contact Form

This template contains a fully working contact form. All you need to do is to replace one line of code. So, in order to receive messages you need to edit contact.php file from php folder and type in your own email address. You could change the subject as well. 

// Enter your email address below.
// Example $to_address = "bruce.wayne@yourdomain.com";
$to_address = "you_email_address@domain.com"; 

--- more content ---

// Also, you can change the value of the $subject variable to whatever you like
$subject = "$name has contacted you via your Website";

Warning! Please ensure that PHP mail() function is correctly configured on your server.

Flickr Feed

If you want to use flickr feed widget, you'll have to change one line of code. Go to script.js from js folder and in "Plugin configurations" section you'll find the flickr feed script. Change the id property to your flickr ID. See an example below.

var flickrData = {
    limit: 6,
    qstrings: {
        // You have to put your flickr ID here
        id: '52617155@N08'
    },
    itemTemplate: '<li><a href="{{link}}" target="_blank"><img src="{{image_s}}" alt="{{title}}" /></a></li>'
};
